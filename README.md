## 项目介绍

这是一个为 Spring Boot 新手设计的练手项目，实现了基本的图书管理功能：
- 图书的增删改查
- 按作者搜索图书
- 使用 Spring Data JPA 操作数据库

## 技术栈

- **核心框架**: Spring Boot 3.x
- **数据库**: MySQL
- **数据访问**: Spring Data JPA
- **构建工具**: Maven
- **API 风格**: RESTful

## 项目特点（适合新手学习）

1. 清晰的 MVC 分层结构
2. 详细的代码注释
3. 使用 Spring Data JPA 简化数据库操作
4. 完整的 REST API 设计
5. 简单易懂的业务逻辑
